# CodeFlowMind
CodeFlowMind는 캐치마인드 웹 게임 1인 프로젝트(CodeFlowMind)입니다.
CodeFlowMind는 웹으로 플레이할 수 있는 캐치마인드 게임이며, 게임을 즐기기 위한 방 생성, 방 입장이 가능합니다.
인 게임에서 본격적인 게임을 구성하는 방장 기능, 그리기 기능, 채팅 기능 등등이 구현되어 있습니다.
